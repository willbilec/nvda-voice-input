Groq Voice Dictation

Groq Voice Dictation adds push-to-toggle voice dictation to NVDA.

Usage

  • Press NVDA+Shift+V to start recording.
  • Press the same gesture again to stop and send the recording.
  • If silence detection is enabled, recording stops automatically after the configured quiet period.
  • Press the gesture twice in quick succession (within half a second) to force-stop the add-on. Use this if it ever appears stuck during transcription, cleanup, or while waiting on the readback/confirm window. The audio is discarded and any pending insert is cancelled.

Settings

Open NVDA Settings and choose Groq Voice Dictation to configure your Groq API key, default microphone, cleanup mode, silence detection, feedback, and paste fallback.

The settings panel includes a Get API key button that first shows setup instructions in a standard dialog and then opens the Groq keys page. Use NVDA's installed add-ons dialog for the add-on documentation/help entry.

Notes

  • The add-on does not ship with an API key.
  • Audio is recorded locally, sent to Groq for transcription, optionally cleaned up with a second Groq request, and then inserted into the focused control.
